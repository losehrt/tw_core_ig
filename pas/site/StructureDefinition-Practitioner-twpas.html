Resource Profile: 醫事人員-Practitioner TWPAS

Official URL: https://nhicore.nhi.gov.tw/pas/StructureDefinition/Practitioner-twpas Version: 1.0.6
Active as of 2025-07-30 Computable Name: PractitionerTWPAS

此醫事人員-Practitioner TWPAS Profile說明本IG如何進一步定義臺灣核心-健康照護服務提供者(TW Core Practitioner) Profile以呈現癌藥事前審查之申請醫師、簽發影像報告醫師、簽發癌症分期報告醫師、檢查報告醫師、簽發檢驗(查)報告醫事人員及評估項目醫事人員基本資料

Usages:

  • Use this Profile: 事前審查-Bundle TWPAS
  • Refer to this Profile: 事前審查-Claim TWPAS, 影像報告-DiagnosticReport Image TWPAS, 檢查報告-DiagnosticReport TWPAS, 癌症分期量表-Observation Cancer Stage TWPAS... Show 2 more, 檢驗檢查-Observation Laboratory Result TWPAS and 病人狀態評估-Observation Patient Assessment TWPAS
  • Examples for this Profile: Practitioner/pra-min
  • CapabilityStatements using this Profile: 臺灣癌藥事前審查-用戶端(TWPAS Client) and 臺灣癌藥事前審查-伺服端(TWPAS Server)
